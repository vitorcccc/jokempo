# CP1 - HandsOn 02 - Jokempo

## Integrante
- Vitor Couto Victorino - RM 554965

## Estrutura do projeto
- Jokempo.Core
  - Contém a lógica do jogo, regras, estatísticas e jogador.
- Jokempo.WinForms
  - Contém a estrutura da interface gráfica em Windows Forms.

## Funcionalidades
- Cadastro de jogador
- Jogadas de pedra, papel e tesoura
- Resultado de cada rodada
- Estatísticas de vitórias, derrotas e empates
- Alteração de jogador com reset das estatísticas
